# Часы на газоразрядных индикаторах (Nixie Clock)

Многофункциональная модульная прошивка для часов на газоразрядных лампах ИН-12, ИН-14, ИН-8 и аналогичных.  
**Платформа:** ATmega8 / ATmega328 @ 8 МГц · **Среда:** PlatformIO · **Язык:** C++

## 🌟 Основные возможности

| Функция | Описание |
|---|---|
| **Часы и дата** | DS1307 (I2C), формат 24 ч, периодический показ даты |
| **2 будильника** | Независимые, с прерывистым звуковым сигналом |
| **Ежечасный сигнал** | «Кукушка» — короткий звуковой сигнал каждый час |
| **Ночной режим** | По расписанию или автоматически по датчику света |
| **Антиотравление (Etching)** | Перебор цифр 0-9 каждые 10 минут для защиты катодов |
| **Коррекция хода** | Компенсация дрейфа кварца DS1307 (±99 сек / N дней) |
| **4 уровня яркости** | ШИМ через микрозадержки (0, 1, 2, 3) |
| **Подсветка (LED)** | Включение/отключение светодиодной подсветки |
| **EEPROM** | Все настройки сохраняются и восстанавливаются при включении |
| **Аналоговая клавиатура** | 2 кнопки через один АЦП-пин (резисторный делитель) |

## 🔧 Аппаратные требования

### Минимальная конфигурация

| Компонент | Описание |
|---|---|
| **MCU** | ATmega8 или ATmega328P (8 МГц, внутренний/внешний кварц) |
| **RTC** | Модуль DS1307 с батарейкой CR2032 |
| **Индикаторы** | 6 газоразрядных ламп (ИН-12, ИН-14, ИН-8 и т.д.) |
| **Дешифраторы** | 2 × К155ИД1 (SN74141) — BCD → десятичный декодер |
| **Высоковольтный источник** | DC-DC преобразователь ~170 В для анодов |
| **Кнопки** | 2 шт. + резисторный делитель для АЦП |
| **Пищалка** | Пьезоизлучатель (активный или пассивный) |
| **Фоторезистор** | Для автоматического ночного режима (опционально) |
| **LED** | Подсветка ламп (опционально) |

### Распиновка (по умолчанию, настраивается в `hardware.h`)

```
BCD-дешифратор (верхний банк, лампы 1-3):
  PD6, PD7, PB0, PB1

BCD-дешифратор (нижний банк, лампы 4-6):
  PB7, PD3, PD2, PB6

Управление анодами (попарно):
  PC0 — пара ламп 1+4
  PC1 — пара ламп 2+5
  PC2 — пара ламп 3+6

Десятичные точки:
  PD4 — верхний банк
  PC3 — нижний банк

Периферия:
  PD5 — пищалка (buzzer)
  PB2 — подсветка (LED)
  ADC6 — кнопки (аналоговый вход)
  ADC7 — датчик освещенности
```

## 📦 Описание модулей

### `display` — Динамическая индикация
Управляет 6 газоразрядными лампами через мультиплексирование по 3 парам. Каждая пара подключена к общему аноду. Яркость регулируется длительностью задержки между переключениями (4 уровня: 1/266/533/800 мкс). Включает функцию **etching** — антиотравление катодов.

### `ds1307` — Часы реального времени
Чтение/запись даты и времени через I2C. BCD-кодирование. Функция `getMaxMonthDay()` для корректного определения количества дней в месяце (с учётом високосного года).

### `buttons` — Обработка кнопок
Аналоговая клавиатура через АЦП. Определяет 4 типа событий: `BTN_SHORT_1`, `BTN_SHORT_2`, `BTN_LONG_1`, `BTN_LONG_2`. Антидребезг — 60 мс. Длинное нажатие — от 1.3 сек (с автоповтором).

### `alarm` — Будильники и ежечасный сигнал
Два независимых будильника с защитой от повторного срабатывания в тот же день. Ежечасный сигнал с автоматическим подавлением при совпадении с будильником. LED-мигание при срабатывании.

### `beeper` — Звуковой генератор
Неблокирующий генератор прерывистых звуковых паттернов. Настраиваемые параметры: количество бипов, длительность активного/неактивного состояния, пауза между сериями.

### `night_mode` — Ночной режим
Три режима работы:
- **0 (OFF):** Выключен
- **1 (SCHEDULE):** По расписанию (час начала → час окончания)
- **2 (AUTO):** По датчику освещённости с настраиваемым порогом

При активации: яркость → минимальная, LED → выключен.

### `time_correction` — Коррекция хода
Компенсирует накопленную погрешность кварца DS1307. Настраивается: направление (вперёд/назад), величина (0-99 сек), интервал (0-99 дней). Коррекция выполняется в безопасном диапазоне минут (3-57) для избежания проблем на границе часов.

### `millis` — Системный таймер
Использует Timer0 с прескейлером /8. Обеспечивает неблокирующее отслеживание времени для всех модулей (мигание, debounce, beeper, etching, transition-анимации).

## 📖 Инструкция пользователя

### Управление

Часы управляются **двумя кнопками**. Каждая кнопка поддерживает **короткое** и **длинное** (удержание > 1.3 сек) нажатие.

### Основной режим (показ времени)

| Действие | Что произойдёт |
|---|---|
| **Кнопка 1 — коротко** | Вкл/выкл ежечасного сигнала («кукушка»). При ночном режиме — подтверждение миганием LED |
| **Кнопка 2 — коротко** | Переключение яркости по кругу: 0 → 1 → 2 → 3 → 0. При ночном режиме — мигание LED |
| **Кнопка 1 — длинно** | **Вход в меню настроек** |
| **Кнопка 2 — длинно** | Вкл/выкл подсветки LED (не работает в ночном режиме) |
| **Любая кнопка** | Отключает звонящий будильник |

### Режим настроек (меню)

| Действие | Что произойдёт |
|---|---|
| **Кнопка 1 — коротко** | Увеличить значение (+1) |
| **Кнопка 2 — коротко** | Уменьшить значение (−1) |
| **Кнопка 1 — длинно** | Сохранить и перейти к следующему пункту |

> **Автовозврат:** Если не нажимать кнопки 10 секунд, часы автоматически вернутся в рабочий режим.

> **Визуальная индикация:** Редактируемая пара цифр мигает. При переходе к новой группе настроек на центральных лампах (3-4) на 1.5 секунды загорается **номер группы** (1-8).

## 📋 Подробное описание меню настроек

Меню состоит из **22 пунктов**, организованных в **8 групп**. Навигация — последовательная: длинное нажатие Кнопки 1 переходит к следующему пункту. После последнего пункта — возврат в основной режим.

### Группа 1 — Настройка времени

На дисплее отображается: `ЧЧ:ММ:СС`

| Шаг | Что настраивается | Мигает | Диапазон | Особенности |
|---|---|---|---|---|
| 1.1 | Секунды | Лампы 5-6 | — | Нажатие +/− **обнуляет** секунды. Если секунды ≥ 30, минуты автоматически округляются вверх |
| 1.2 | Минуты | Лампы 3-4 | 0 — 59 | — |
| 1.3 | Часы | Лампы 1-2 | 0 — 23 | — |

### Группа 2 — Будильник 1

На дисплее: `ЧЧ:ММ:ВВ` (ВВ = вкл/выкл)

| Шаг | Что настраивается | Мигает | Диапазон |
|---|---|---|---|
| 2.1 | Активность | Лампы 5-6 | 0 (выкл) / 1 (вкл) |
| 2.2 | Минуты | Лампы 3-4 | 0 — 59 |
| 2.3 | Часы | Лампы 1-2 | 0 — 23 |

> Десятичная точка у лампы 5 показывает состояние будильника 1.

### Группа 3 — Будильник 2

Аналогично группе 2, но для второго будильника. Точка у лампы 6.

### Группа 4 — Настройка даты

На дисплее: `ДД:ММ:ГГ`

| Шаг | Что настраивается | Мигает | Диапазон |
|---|---|---|---|
| 4.1 | Год | Лампы 5-6 | 0 — 99 (= 2000-2099) |
| 4.2 | Месяц | Лампы 3-4 | 1 — 12 |
| 4.3 | День | Лампы 1-2 | 1 — макс. для месяца |

> Максимальное число дней автоматически пересчитывается при смене месяца (учитывается високосный год).

### Группа 5 — Периодический показ даты

На дисплее: `НН:КК:ВВ` (НН = секунда начала, КК = секунда конца, ВВ = вкл/выкл)

| Шаг | Что настраивается | Мигает | Диапазон | По умолчанию |
|---|---|---|---|---|
| 5.1 | Показ вкл/выкл | Лампы 5-6 | 0 / 1 | 1 (вкл) |
| 5.2 | Секунда окончания | Лампы 3-4 | 0 — 59 | 53 |
| 5.3 | Секунда начала | Лампы 1-2 | 0 — (сек. оконч.) | 51 |

> **Пример:** При значениях 51/53 — каждую минуту с 51-й по 53-ю секунду вместо времени будет показана дата.

### Группа 6 — Ночной режим

На дисплее: `НЧ:КЧ:ТТ` (НЧ = час начала, КЧ = час конца, ТТ = тип)

| Шаг | Что настраивается | Мигает | Диапазон | Описание |
|---|---|---|---|---|
| 6.1 | Тип режима | Лампы 5-6 | 0 — 2 | 0=выкл, 1=расписание, 2=автомат |
| 6.2 | Час окончания | Лампы 3-4 | 0 — 23 | Когда часы «просыпаются» |
| 6.3 | Час начала | Лампы 1-2 | 0 — 23 | Когда часы «засыпают» |

**Логика расписания:**
- `Начало > Конец` (напр. 22→07): ночной режим через полночь — активен с 22:00 до 07:00
- `Начало < Конец` (напр. 01→06): активен с 01:00 до 06:00
- `Начало == Конец`: ночной режим **24/7** (используйте с осторожностью!)

### Группа 7 — Порог датчика света

На дисплее: `00:00:ПП` (ПП = порог)

| Шаг | Что настраивается | Мигает | Диапазон | По умолчанию |
|---|---|---|---|---|
| 7.1 | Чувствительность | Лампы 5-6 | 0 — 50 | 25 |

> **Как это работает:** Чем выше значение, тем темнее должно быть для активации ночного режима. Значение 0 — режим срабатывает при любом освещении. Во время настройки LED-подсветка включается/выключается в реальном времени, показывая текущий порог.

### Группа 8 — Коррекция хода

На дисплее: `ИН:ЗН:НП` (ИН = интервал, ЗН = значение, НП = направление)

| Шаг | Что настраивается | Мигает | Диапазон | Описание |
|---|---|---|---|---|
| 8.1 | Направление | Лампы 5-6 | 0 / 1 | 1=часы спешат (убавить), 0=отстают (прибавить) |
| 8.2 | Значение (сек.) | Лампы 3-4 | 0 — 99 | На сколько секунд корректировать |
| 8.3 | Интервал (дни) | Лампы 1-2 | 0 — 99 | Раз в сколько дней |

> **Пример:** Направление=1, Значение=5, Интервал=10 → каждые 10 дней часы автоматически отнимут 5 секунд.  
> **Отключение:** Установите значение или интервал в 0.

## 💡 Индикация состояния

### Десятичные точки (в основном режиме)

| Точка | Что показывает |
|---|---|
| Точка лампы 5 (верхняя) | Будильник 1 активен |
| Точка лампы 6 (нижняя) | Будильник 2 активен **или** ежечасный сигнал активен |

### Поведение LED-подсветки

| Ситуация | Поведение LED |
|---|---|
| Обычный режим | Включена/выключена (по настройке пользователя) |
| Ночной режим | Выключена |
| Срабатывание будильника | Быстрое мигание (5 вспышек) |
| Нажатие кнопки в ночном режиме | Короткое мигание (подтверждение) |

### Антиотравление (Etching)

Каждые **10 минут** на всех лампах одновременно быстро (170 мс/шаг) прокручиваются цифры 0→1→2→...→9. Это предотвращает «отравление» катодов, которые редко зажигаются, и продлевает срок службы ламп. Во время анимации кнопки заблокированы.

## ⚙ Конфигурация прошивки

### Аппаратная конфигурация (`include/hardware.h`)

**Это единственный файл, который нужно менять при переносе на другую плату.** Все пины, порты и каналы АЦП настраиваются здесь:

```cpp
#define PIN_LED              PB2       // Пин подсветки
#define PIN_BUZZER           PD5       // Пин пищалки
#define ADC_CHANNEL_BUTTONS  6         // Канал АЦП для кнопок
#define ADC_CHANNEL_LIGHT    7         // Канал АЦП для фоторезистора
#define INIT_DDRD            0xFC      // Регистр направления порта D
#define INIT_DDRB            0xC7      // Регистр направления порта B
#define INIT_DDRC            0x0F      // Регистр направления порта C
```

### Настройки меню (`src/menu.h`)

```cpp
#define MENU_RETURN_TIMEOUT        10     // Автовозврат через N секунд
#define MENU_TRANSITION_ENABLED    1      // 1=показывать номер группы, 0=нет
#define MENU_TRANSITION_DURATION_MS 1500  // Длительность анимации перехода (мс)
```

### Настройки дисплея (`lib/display/display.h`)

```cpp
#define BRIGHTNESS_MAX       3        // Максимальный уровень яркости
#define ETCHING_DURATION_MS  170      // Скорость перебора цифр (мс/шаг)
#define ETCHING_INTERVAL_MIN 10       // Интервал антиотравления (минуты)
```

### Настройки кнопок (`lib/buttons/buttons.h`)

```cpp
#define BTN_THRESHOLD_BUTTON2   581   // Порог АЦП для кнопки 2
#define BTN_THRESHOLD_BUTTON1   770   // Порог АЦП для кнопки 1
#define BUTTON_DEBOUNCE_MS      60    // Антидребезг (мс)
#define BUTTON_LONG_PRESS_MS    1300  // Порог длинного нажатия (мс)
```

> **Калибровка кнопок:** Подключите плату, считайте значение АЦП при нажатии каждой кнопки (через UART или отладчик) и подставьте пороговые значения.

## 🛠 Сборка и прошивка

### Необходимое ПО

1. **Visual Studio Code** + расширение **PlatformIO IDE**
2. Программатор **USBasp** (или другой поддерживаемый PlatformIO)

### Пошаговая инструкция

1. Установите VSCode и PlatformIO
2. Откройте папку проекта (`clock/`)
3. Проверьте `platformio.ini`:
   ```ini
   [env:atmega8]
   platform = atmelavr
   board = ATmega8
   board_build.mcu = atmega8
   board_build.f_cpu = 8000000L
   upload_protocol = usbasp
   build_flags = -Os
   ```
   Для **ATmega328P** замените:
   ```ini
   [env:atmega328]
   platform = atmelavr
   board = ATmega328P
   board_build.mcu = atmega328p
   board_build.f_cpu = 8000000L
   upload_protocol = usbasp
   build_flags = -Os
   ```
4. Нажмите **Build** (✓) в панели PlatformIO
5. Подключите программатор → нажмите **Upload** (→)

### Настройка фьюзов (ВАЖНО!)

Убедитесь, что фьюзы MCU настроены на **8 МГц**:

**ATmega8:**
- Low Fuse: `0xE4` (внутренний RC 8 МГц) или `0xFF` (внешний кварц 8 МГц)
- High Fuse: `0xD9`

**ATmega328P:**
- Low Fuse: `0xE2` (внутренний RC 8 МГц) или `0xFF` (внешний кварц)
- High Fuse: `0xD9`
- Extended Fuse: `0xFF`

> ⚠️ **Внимание:** Неправильная установка фьюзов может заблокировать микроконтроллер! Перепроверьте значения с помощью [AVR Fuse Calculator](https://www.engbedded.com/fusecalc/).

## 🔍 Устранение неполадок

| Проблема | Возможная причина | Решение |
|---|---|---|
| Лампы не светятся | Нет высокого напряжения на анодах | Проверьте DC-DC преобразователь (~170 В) |
| Показывают мусор | Неверная распиновка дешифратора | Проверьте BCD-пины в `hardware.h` |
| Время сбрасывается при выключении | Батарейка DS1307 разряжена | Замените CR2032 |
| Время отображается некорректно | I2C не работает | Проверьте подтягивающие резисторы 4.7 кОм на SDA/SCL |
| Кнопки не реагируют | Неверные пороги АЦП | Откалибруйте `BTN_THRESHOLD_*` в `buttons.h` |
| Кнопки «дребезжат» | Слишком низкий debounce | Увеличьте `BUTTON_DEBOUNCE_MS` |
| Ночной режим не выключается | `hourStart == hourStop` | Установите разные значения начала и конца |
| Лампы тускнеют со временем | Отравление катодов | Etching работает автоматически; проверьте, что `ETCHING_INTERVAL_MIN` не слишком велик |
| Часы спешат/отстают | Дешёвый кварц DS1307 | Настройте коррекцию хода (Группа 8 меню) |

## 📄 Значения EEPROM по умолчанию

При первой прошивке (или при повреждении EEPROM) используются следующие значения:

| Параметр | По умолчанию |
|---|---|
| Яркость | 3 (максимум) |
| Подсветка LED | Включена |
| Ежечасный сигнал | Включен |
| Будильник 1 | Выключен, 07:00 |
| Будильник 2 | Выключен, 08:00 |
| Показ даты | Включен, 51-53 сек |
| Ночной режим | Авто (тип 2), 22:00-07:00, порог 25 |
| Коррекция хода | Вперёд, 0 сек, 0 дней (выключена) |
